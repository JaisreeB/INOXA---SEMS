# ⚡ INOXA – Smart Energy Management System (SEMS)

A modern Smart Energy Management System (SEMS) dashboard that enables homeowners to monitor, analyze, and optimize electricity consumption through an intuitive, responsive web interface.

---

## Overview

INOXA is a smart home energy management dashboard developed to visualize energy consumption and provide actionable insights for improving energy efficiency.

The application presents energy statistics, device monitoring, room-wise consumption, automation controls, and analytical charts within a clean, responsive interface suitable for desktops and mobile devices.

---

## Features

* 📊 Real-time Energy Dashboard
* ⚡ Power Consumption Monitoring
* 🏠 Room-wise Energy Usage
* 📈 Interactive Charts and Analytics
* 🤖 AI Energy Assistant (UI)
* 🔔 Smart Alerts & Notifications
* 🌙 Dark / Light Theme Support
* 📱 Fully Responsive Design
* 🎛 Smart Device Control Interface
* 📅 Automation Scheduling Interface

## How to Run

1. Clone the repository.

```bash
git clone https://github.com/YOUR_USERNAME/INOXA-SEMS.git
```

2. Open the project folder.

3. Launch `index.html` in your browser.

**Recommended:** Use the **Live Server** extension in Visual Studio Code for the best development experience.
